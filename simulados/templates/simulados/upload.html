Padrão de Importação

O sistema utiliza a primeira linha do seu arquivo para mapear os dados. Seu arquivo Excel ou Bloco de Notas deve conter exatamente estas colunas:

  • assunto
  • enunciado
  • a
  • b
  • c
  • d
  • correta (A, B, C ou D)
  • justificativa
Sistema Anti-Duplicidade: Se a questão já existir no banco, ela será ignorada automaticamente.
